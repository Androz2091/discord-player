## QueueRepeatMode

The queue repeat mode. This can be one of:
- OFF
- TRACK
- QUEUE
- AUTOPLAY

| Property | Type | Value |
| ----------- | ----------- | ----------- |
| AUTOPLAY | 3 | N/A |
| OFF | 0 | N/A |
| QUEUE | 2 | N/A |
| TRACK | 1 | N/A |
